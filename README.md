cli to setup boilerplate

## Usage

1. use template from <https://github.com/tjx666/awesome-vscode-extension-boilerplate>
2. `git clone <forked-git-repository>`
3. `cd <forked-git-repository-name>`
4. `npx setup-boilerplate`
